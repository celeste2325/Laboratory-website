# LabWebsite
This is a simple laboratory website where users can find information about the laboratory and its services.

this mandatory excersice pretends to manage variables, arrays, functions, and loops. It is basic javascript without connection to HTML. In this excersice, I simulated a database call to get the laborary services, order them in ascending alphabetical order, and display them. In the future this data will be get from the database and shown on the corresponding page. I also added some functions to simulate how to manage data from form fields and save them to the database. In the future this data will be sent to the database when the user clicks the save button.
